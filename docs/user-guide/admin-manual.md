
# Manual de Administrador - Sistema Puerto López

## 1. Introducción

Este manual describe las funcionalidades administrativas del Sistema de Gestión Turística Puerto López.

## 2. Acceso al Sistema

### 2.1 Inicio de Sesión
1. Navegar a la página de login
2. Ingresar credenciales de administrador
3. Completar verificación CAPTCHA
4. Click en "Iniciar Sesión"

### 2.2 Dashboard Principal
El dashboard proporciona acceso a:
- Gestión de contenido del sitio
- Administración de atracciones turísticas
- Gestión de galería de imágenes
- Perfil de usuario

## 3. Gestión de Contenido

### 3.1 Edición de Secciones
**Hero Section:**
- Título principal del sitio
- Subtítulo descriptivo
- Descripción detallada
- Imagen de fondo

**Footer:**
- Información de contacto
- Dirección física
- Teléfono y email
- Descripción de la empresa

### 3.2 Proceso de Edición
1. Acceder al Dashboard
2. Seleccionar "Editor de Contenido"
3. Elegir la sección a editar
4. Realizar cambios en los campos
5. Click en "Guardar Cambios"
6. Verificar los cambios en el sitio público

## 4. Gestión de Atracciones Turísticas

### 4.1 Crear Nueva Atracción
1. Ir a "Gestión de Atracciones"
2. Click en "Nueva Atracción"
3. Completar información:
   - Nombre de la atracción
   - Descripción detallada
   - Categoría (Playas, Naturaleza, Cultura, etc.)
   - Imagen principal
4. Guardar cambios

### 4.2 Editar Atracción Existente
1. Localizar la atracción en la lista
2. Click en "Editar"
3. Modificar campos necesarios
4. Subir nueva imagen si es necesario
5. Guardar cambios

### 4.3 Gestión de Imágenes
- Formatos soportados: JPG, PNG, WebP
- Tamaño máximo recomendado: 2MB
- Dimensiones recomendadas: 1200x800px
- Texto alternativo obligatorio para accesibilidad

## 5. Galería de Imágenes

### 5.1 Organización por Categorías
- Vida Marina
- Paisajes
- Cultura Local
- Actividades Turísticas

### 5.2 Subida de Imágenes
1. Seleccionar categoría
2. Click en "Subir Imagen"
3. Seleccionar archivo
4. Añadir título y descripción
5. Especificar orden de visualización
6. Guardar

## 6. Mejores Prácticas

### 6.1 Contenido
- Usar lenguaje claro y atractivo
- Verificar ortografía antes de publicar
- Mantener información actualizada
- Incluir llamadas a la acción

### 6.2 Imágenes
- Optimizar tamaño para web
- Usar imágenes de alta calidad
- Incluir texto alternativo descriptivo
- Mantener consistencia visual

### 6.3 SEO
- Usar títulos descriptivos
- Incluir palabras clave relevantes
- Mantener meta descripciones actualizadas
- Verificar enlaces rotos regularmente

## 7. Solución de Problemas

### 7.1 Problemas Comunes
**Error al subir imagen:**
- Verificar tamaño del archivo
- Confirmar formato compatible
- Revisar conexión a internet

**Cambios no se reflejan:**
- Limpiar caché del navegador
- Esperar unos minutos para propagación
- Verificar que se guardaron los cambios

### 7.2 Contacto de Soporte
Para problemas técnicos contactar al equipo de desarrollo con:
- Descripción detallada del problema
- Pasos para reproducir el error
- Capturas de pantalla si es posible
